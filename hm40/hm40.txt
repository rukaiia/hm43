Таблица "courses":
Содержит информацию о курсах, запись имеет уникальный идентификатор id и поле name для названия курса.

Таблица "students":
Хранит данные о студентах
Включает уникальный id для каждого студента, а также поля name (имя студента), start_studing (дата начала обучения) и finished_studing (дата окончания обучения).

Таблица "teachers":
Содержит информацию о преподавателях,
Включает id и поле name для имени преподавателя.

Таблица "group":
Используется для организации студентов в учебные группы по курсам.
Каждая запись включает  id группы, а также поля name, start_date (дата начала занятий), finish_date (дата окончания занятий), course_id (связь с таблицей "courses"), teacher_id (связь с таблицей "teachers" ) и student_id (связь с таблицей "students").
